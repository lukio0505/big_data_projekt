# 🌤️ Dashboard Analityczny - Pogoda w Polsce (2023)

Projekt zaliczeniowy z przedmiotu zarządzanie big data.
Aplikacja pobiera rzeczywiste, historyczne dane z darmowego API Open-Meteo, czyści je i wizualizuje.

**Funkcjonalności:**
- 5 typów wykresów (Liniowy, Słupkowy, Scatter, Histogram, Mapa).
- 3 widgety filtrujące (zakres dat, multiselect miast, selectbox rodzaju pogody).

**Uruchomienie lokalnie:**
`pip install -r requirements.txt`
`streamlit run app.py`
